# Описание продукта

## Проблема

Команда технической поддержки сталкивается с типичными проблемами:
- Ручное распределение дежурств отнимает время и приводит к ошибкам
- Инциденты из разных источников (почта, Jira) легко пропустить
- Нет единой точки входа для справочной информации по контурам
- Статистика ведётся вручную или не ведётся вовсе

## Решение

**Support Admin Bot** — Telegram-бот, который автоматизирует управление дежурствами, мониторит источники инцидентов и предоставляет справочную информацию.

---

## Глоссарий

| Термин | Определение |
|--------|-------------|
| **Контур** | Изолированная среда или подсистема (например, `etl.corp`, `bi.prod`, `dwh.test`) |
| **Дежурный** | Сотрудник, ответственный за обработку инцидентов в свою смену |
| **Инцидент** | Событие, требующее реакции поддержки (письмо, тикет в Jira) |

---

## Роли (Акторы)

| Роль | Описание | Примеры действий |
|------|----------|------------------|
| **Админ** | Настраивает бота, управляет расписанием и сотрудниками | Добавить сотрудника, задать расписание, настроить контур |
| **Дежурный** | Получает уведомления, обрабатывает инциденты | Получить напоминание, посмотреть новые тикеты |
| **Сотрудник** | Просматривает справку, запрашивает информацию | `help etl.corp`, запросить статистику |

---

## Дорожная карта

### Фаза 1 — MVP (Базовый бот)
- [ ] Регистрация бота в Telegram-чате поддержки
- [ ] Добавление/удаление сотрудников (админом)
- [ ] Ручное назначение дежурств на дату
- [ ] Утреннее напоминание дежурному о смене

### Фаза 2 — Автоматизация дежурств
- [ ] Автоматическое распределение дежурств по расписанию
- [ ] Учёт отпусков и отсутствий сотрудников
- [ ] Возможность поменяться сменами

### Фаза 3 — Мониторинг источников
- [ ] Интеграция с почтой (IMAP/API)
- [ ] Интеграция с Jira (REST API)
- [ ] Уведомления дежурному о новых инцидентах/дефектах

### Фаза 4 — Отчётность
- [ ] Ведение статистики по инцидентам в БД
- [ ] Агрегация данных по контурам и периодам
- [ ] Экспорт отчётов в Excel по запросу

---

## Справочная система (Контуры)

Бот предоставляет справку по контурам через команду `help <контур>`.

**Пример:** `help etl.corp`

**Справка содержит:**
- Владелец экспертизы (ответственный)
- Ссылка на документацию
- Контакты для эскалации

---

## Out of Scope (НЕ входит в проект)

- Веб-интерфейс (только Telegram)
- Поддержка других мессенджеров (Slack, Teams)
- Автоматическое решение инцидентов (только уведомления)
- Интеграция с системами on-call (PagerDuty, OpsGenie)
