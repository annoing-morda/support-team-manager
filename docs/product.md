# Описание продукта

## Общее описание

Support Admin Bot — это Telegram-бот, администратор группы технической поддержки.

## Функционал

### 1. Управление дежурствами

- Бот добавляется в чат с сотрудниками поддержки
- Распределяет дежурства между сотрудниками
- Утром напоминает дежурным о том, что они дежурят
- Учитывает отпуска и отсутствия сотрудников

### 2. Мониторинг источников

- Мониторинг почты
- Мониторинг Jira
- Уведомления дежурному о новых дефектах/инцидентах, требующих действий

### 3. Статистика

- Ведение статистики по инцидентам
- Выгрузка данных в базу данных
- Экспорт в Excel по запросу

### 4. Контуры поддержки

- Поддержка нескольких контуров
- Команда `help` по контуру (например, `help etl.corp`)
- Справка содержит: владельца экспертизы, ссылку на документацию и т.д.
