# Support Admin Bot

Telegram-бот для администрирования группы технической поддержки.

## Возможности

- Распределение дежурств между сотрудниками
- Напоминания дежурным о смене
- Учёт отпусков и отсутствий
- Мониторинг источников (почта, Jira)
- Уведомления о новых инцидентах/дефектах
- Статистика по инцидентам (БД, экспорт в Excel)
- Поддержка нескольких контуров с командами справки
